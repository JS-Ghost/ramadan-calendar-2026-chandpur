A premium, modern web app for the Ramadan calendar in Chandpur, Bangladesh. Inspired by Google Calendar and Apple Calendar, this app features a clean, responsive interface, live countdowns, and elegant Islamic design.

## Features

- **Professional Calendar Grid:** 7-column monthly view, dates placed according to the real 2026 calendar.
- **Ramadan Day Details:** Sehri & Iftar times for each day, highlighted Fridays and 27th Ramadan.
- **Live Clock & Countdown:** Smart countdown to Sehri/Iftar, fasting progress bar.
- **Responsive Design:** Desktop grid + side panel, mobile stacked cards.
- **Dark/Light Mode:** Theme toggle with localStorage persistence.
- **Microinteractions:** Ripple effects, smooth transitions, glassmorphism cards.
- **LocalStorage:** Remembers theme and last selected date.

## Usage

1. **Open index.html** in your browser.
2. **Select a date** to view details and progress.
3. **Toggle theme** for dark/light mode.
4. **Responsive:** Works on desktop, tablet, and mobile.

## Tech Stack

- HTML5, CSS3 (with CSS variables)
- JavaScript (modular, production-ready)
- No frameworks, no dependencies
